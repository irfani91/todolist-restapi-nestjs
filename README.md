## Description

To do list with Rest API with simple functional such as :
- CRUD
- Send Email
- JWT Auth
- Relationanl 
- Prisma Migration
- Simple Unit Testing
- Compodoc
- Docker 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
